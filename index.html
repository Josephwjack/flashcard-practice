JavaScript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

A special symbol used to perform operations on operands (values and variables).

Variables

Anything that can vary. A variable stores the data value that can be changed later on.

Variable naming conventions

Always use camel case and describe stored value.

Functions

A set of statements that performs a task or calculates a value.

Methods

Actions that can be performed by objects.

Arguments

An object which is local to the function.

Parameters

A named variable passed into a function.
